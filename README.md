# Who is smarter
Made by Isac Larsson 2026-03-9

# What it does
Its a website where you can decide whos the smartest of the girls
in my class.

# How it works
It uses front end development to display two clickable buttons. In
Java-script i made a algorithm that randomize a index value in a list
to decide what person the button should display. I also built a algorithm
to prevent that the same names displays on the buttons.
